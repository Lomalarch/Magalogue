Vous avez plusieurs options pour personnaliser l’apparence de votre blog.

En-tête

Vous pouvez indiquer l'URL d'un logo à utiliser à la place du logo par défaut. Si l'URL ne débute pas par un / il s'agira d'une URL relative au dossier img du thème (mettre MagalogueBanner.png revient par exemple à utiliser le logo par défaut). Videz ce champ pour revenir à la configuration par défaut.

Vous pouvez, si « Menu Simple » est installé et actif, cliquer sur le lien indiqué pour paramétrer le menu affiché en entête.

Couleur des liens

Dans cette section, vous pouvez choisir la couleur de mise en valeur parmi les quatre options proposées.

Liens vers les médias sociaux

Les étiquettes s'afficheront en haut de l'écran sous forme d'imagettes sur les écrans larges et en bas de la page sous forme de boutons sur les tout petits écrans. Toutes les images dont le nom de fichier correspond au schéma sticker-*.png et se trouvant dans le sous-répertoire img du thème sont listées. Vous pourrez indiquer pour chacune un libellé et l'URL correspondante. (Laissez l'un de ces deux champs vide pour ne pas l'afficher.)

Vous pouvez également modifier l'ordre d'affichage par tiré-lâché si javascript est actif et que vous ne l'avez pas interdit dans vos préférences. Si javascript est désactivé, un champ supplémentaire permet d'indiquer l'ordre d'affichage (de 1 à n).

Listes de billets sur la page d’accueil

La page d’accueil est constituée d’un diaporama suivi d’une liste de billets. Vous pouvez choisir selon quels critères sont choisis les billets affichés dans les deux zones :

Sélectionné
affichera les billets marqués comme Sélectionnés dans la liste des billets
Derniers billets
affichera les billets du plus récents au plus ancien
Catégories de premier niveau
affichera pour chaque catégorie de premier niveau (sans catégorie parent) le dernier billet publié
Toutes les catégories
affichera le dernier billet de chaque catégorie

Pour les options Sélectionné et Derniers billets, il est possible de choisir le nombre de billets à afficher. Par défaut, on affichera cinq publications dans le diaporama et neuf dans la liste des billets.
